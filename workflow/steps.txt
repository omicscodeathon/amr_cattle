Steps:
1. Fetch data from NCBI and mg-rast using  python script custom
2. Quality control of the reads using Trimmomatic
3. Assemmbly using SqueezeMeta
4. Annotation (ORFs,CDS) using CAT, SqueezeMeta
5. Taxonomic assignment using diamond and SqueezeMeta 
6. Functional Annotation(KEGG,eggNOG,CAZy) using Blast and SqueezeMeta
7. Alpha and Beta diversity in R and SqueezeMeta 
8. Identification of AMR genes using Resfinder and rgi
9. Statistical analyses in R
