Steps:
1. Fetch data from NCBI and mg-rast using  python script custom
2. Quality control of the reads using Trimmomatic
3. Assemmbly using SqueezeMeta and YAMP
4. Annotation (ORFs,CDS) using CAT, SqueezeMeta
5. Taxonomic assignment using YAMP and SqueezeMeta 
6. Functional Annotation(KEGG,eggNOG,CAZy) using Blast, YAMP and SqueezeMeta
7. Gene Prediction using YAMP and SqueezeMeta
8. Alpha and Beta diversity in R and SqueezeMeta 
9. Identification of AMR genes using Resfinder, Platon, Abricate e.t.c
10. Statistical analyses in R
